# business-context

Kennisbronnen die `/ad-scripts` (en deels `/ad-research`) automatisch inladen.
Deze bestanden bevatten geen secrets en mogen wél in git — ze zijn onderdeel van
de overdracht. Vul ze met de echte business-data voordat je scripts genereert.

Verwachte bestanden:

- `product-catalog.md` — actieve producten, prijzen, offers
- `advertising-strategy.md` — funnel-mechanismes, creative angles, awareness levels
- `customer-journey.md` — klantpad, pijnpunten, taalgebruik per fase

Tone of voice (NL: warm/empathisch, "begrip voor je hond"; EN: zelfde warmte,
iets directer — Jess' persoonlijke stijl) hoort ook hier thuis, of als aparte
`tone-of-voice.md`.
